# Advanced Path (1600-2000)

Deepen your understanding with complex positions, exceptions, and strategic themes.

**Prerequisites:** [Beginner Path](/study-paths/beginner) and [Intermediate Path](/study-paths/intermediate).

## Learning Path

1. **[Bishop vs knight](/minor-piece-endgames/bishop-vs-knight)**
   Neither piece is universally superior — the position determines which is better, depending on pawn structure and openness.

2. **[Opposite-colour bishops](/minor-piece-endgames/bishop-opposite-color)**
   These endings have strong drawing tendencies but are not automatic draws — learn when the attacker can still win.

3. **[Fortresses](/strategic-themes/fortresses)**
   Defensive setups that cannot be broken despite material disadvantage — recognizing them can save lost positions.

4. **[Two weaknesses](/strategic-themes/principle-of-two-weaknesses)**
   Against a single weakness the defender can hold — creating a second weakness stretches the defense and often wins.

5. **[Do not hurry](/strategic-themes/do-not-hurry)**
   In endgames with a clear advantage, patience is key — premature action can throw away wins.

6. **[Zugzwang](/strategic-themes/zugzwang)**
   A position where any move worsens your situation — learn to recognize and exploit this powerful theme.

7. **[Stalemate resources](/strategic-themes/stalemate-resources)**
   Defensive tricks using stalemate threats to save positions that otherwise seem lost.
